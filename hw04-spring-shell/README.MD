# Текущее задание (ДЗ №4):

## Переписать интерфейс приложения для тестирования студентов с использованием Spring Shell
### Требования к реализации:

* Подключить Spring Shell
* Написать команду, позволяющую проводить тестирование
* Перенести тесты на @SpringBootTest. Компоненты и моки взять из тестового контекста, 
в остальном, за исключением интеграции с файловой системой в тесте дао, это должны быть юнит-тесты
* Локализовывать команды Spring Shell не нужно, а сообщения которые они выводят, нужно
* Данная работа не подразумевает использования Spring Application Events.
